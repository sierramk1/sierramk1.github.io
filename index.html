I completed a semester long project developing a movie recommendation system for Letterboxd, a social platform for film enthusiasts. The project explores three different approaches to creating better recommendations: Singular Value Decomposition (SVD), Non-Negative Matrix Factorization (NMF), and Bayesian Personalized Ranking (BPR). SVD and NMF are matrix factorization techniques that work with explicit feedback (like ratings), while BPR is designed for implicit feedback (such as whether a user has watched a movie). The project also incorporates sentiment analysis of user reviews in an attempt to enhance recommendation accuracy. The final solution combines matrix factorization predictions with sentiment analysis results using a weighted approach, where the weight parameter Î² balances the influence of each method. The performance of these different approaches is evaluated using metrics like Precision@k, Reciprocal Rank, and Normalized Discounted Cumulative Gain (NDCG) to ensure the recommendations are both accurate and relevant to users' preferences.
