OptiLearn: An Interactive Educational Tool for Visualizing Optimization Algorithms

OptiLearn is an interactive web application designed to help students better understand optimization algorithms through visualization and hands-on exploration. Many optimization methods can feel abstract when learned only mathematically, so OptiLearn allows users to interact directly with algorithms by selecting methods, inputting their own functions or data, adjusting parameters, and stepping through each iteration. Animated plots and pseudocode views help bridge the gap between theory and intuition for both one-dimensional and multi-dimensional optimization problems.

The application emphasizes dynamic visualization, robust input validation, and clear user feedback. Each algorithm iteration is precomputed and stored, enabling users to rewind or step forward through animations rendered with Plotly. Because the app is hosted on a serverless platform, all data parsing and interpolation are handled on the frontend, including both cubic spline and piecewise linear interpolation. OptiLearn was built with a strong focus on numerical stability, usability, and educational clarity, making it a practical learning tool for students studying optimization.

Investigating the Relationship Between Low Sleep and BMI

This project examined whether short sleep duration is associated with higher body mass index (BMI) among U.S. adults aged 20 and older. Using nationally representative, self-reported survey data, we conducted exploratory data analysis and multivariable regression modeling to evaluate the relationship between sleep duration and BMI while adjusting for demographic, behavioral, and mental health covariates.

We identified a nonlinear (quadratic) relationship between sleep duration and BMI and implemented a piecewise modeling strategy, focusing on the effects of low sleep (<7 hours per night). Log-transforming BMI improved residual normality and model stability. Results indicated that low sleep was significantly associated with higher BMI (p = 0.006), even after controlling for physical activity, depression, and other covariates. Model diagnostics confirmed appropriate linearity, independence, and low multicollinearity.

Letterboxd Recommendation System

I completed a semester long project developing a movie recommendation system for Letterboxd, a social platform for film enthusiasts. The project explores three different approaches to creating better recommendations: Singular Value Decomposition (SVD), Non-Negative Matrix Factorization (NMF), and Bayesian Personalized Ranking (BPR). SVD and NMF are matrix factorization techniques that work with explicit feedback (like ratings), while BPR is designed for implicit feedback (such as whether a user has watched a movie). The project also incorporates sentiment analysis of user reviews in an attempt to enhance recommendation accuracy. The final solution combines matrix factorization predictions with sentiment analysis results using a weighted approach, where the weight parameter β balances the influence of each method. The performance of these different approaches is evaluated using metrics like Precision@k, Reciprocal Rank, and Normalized Discounted Cumulative Gain (NDCG) to ensure the recommendations are both accurate and relevant to users' preferences.
